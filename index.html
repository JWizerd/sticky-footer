Sticky Footer: Works well with Bootstrap

Notes

  • Make sure that you set a height of 100% and a position relative on the html element. This will make sure that you can fix the footer at the bottom of the screen.
  • Margin Bottom on the body tag should be equal to the fixed height of your footer so when you absolutely position the footer to the bottom of the page your content won't underlap the footer.
  • I got this example from http://getboostrap.com but wanted to add a description about it and wanted to simplify it a tid bit.
